The meta changes constantly as new costumes are released. All recommendations are based on the last banner run shown on each costume card.
If a costume has not had a rerun for a long time, there is a high chance that its recommendation or rating is no longer accurate. Recommendations are only updated after a new rerun banner for that costume has ended.
There are some recommendations for pub costumes with a hint to claim it for 10 tickets. You will not get those tickets unless a banner for that costume is active!
